---
最近尝试了多种Sublime Text 2的插件，打开ST2的时候会提醒某某插件导致运行缓慢。

COM+Shift+P键打开Package Control后，搜索
{% highlight ruby %}
Package Control: Remove Package
{% endhighlight %}
找到要删除的插件就可以自行删除。

如果不是使用Package Control安装的话，就直接删除对应Emmet目录。
